## Support
Discord: https://discord.gg/dge38Gm

## Requirements
- JDK 11
- Maven
- A redis instance

## Start
To start GameAPIs locally run:
```
mvn -B clean package spring-boot:run
```
Your Self-Hosted GameAPIs will be available at 127.0.0.1:8080
